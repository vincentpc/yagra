*********
API 客户端接口
*********

头像访问API
===============

url:
----
   http://example.com/avatar/xxx


HTTP请求方式:
---------
   xxx为账户email的hash值
   
  

请求参数:
-----


* from:

   必选:可选
   
   类型:int   
 
   说明:指定从第几条消息开始列出 (0为最新的第一条消息)
      
            默认为from=0 (即从最新的消息开始列出)


返回数据:
-----
      
      提供原始头像图片,可以另存为(保存原有格式)
